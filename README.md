# 🧠 Skills Repository

A collection of skills for easy reference and usage.

## Available Skills

- **[Zen of Kotlin](./zen-of-kotlin/)**: Idiomatic Kotlin adaptation
  of [James Ward's Zen of James](https://github.com/jamesward/skills). Built for immutable, type-safe code.
- **[Deslopify](./deslopify/)**: Make text more genuine and natural by removing AI tropes and cliches.
  From [glaforge/deslopify](https://github.com/glaforge/deslopify/).
- **[JSpecify Nullness](./jspecify-nullness/)**: Working with JSpecify's nullness annotations.
  From [alexandru/skills](https://github.com/alexandru/skills).
- **[Kotlin Java Library](./kotlin-java-library/)**: Guidelines for creating Kotlin-friendly Java libraries.
  From [alexandru/skills](https://github.com/alexandru/skills).

## Installation

To install as [Junie skills](https://junie.jetbrains.com/docs/agent-skills.html), run:

```bash
./install.sh
```

For more skills, visit [skills.sh](https://skills.sh/).
